# 规则管理

## 查看规则

### 查看所有规则

```bash
# 查看完整规则集
nft list ruleset

# 带句柄号显示（用于删除）
nft -a list ruleset
```

### 查看特定表

```bash
# 查看 nat 表
nft list table ip nat

# 查看 filter 表
nft list table inet filter
```

### 查看特定链

```bash
# 查看 input 链
nft list chain inet filter input

# 带句柄号
nft -a list chain inet filter input
```

## 删除规则

### 方法1：按句柄号删除（推荐）

```bash
# 1. 查看规则和句柄号
nft -a list chain inet filter input

# 输出示例：
# table inet filter {
#   chain input {
#     tcp dport 22 accept # handle 3
#     udp dport 53 drop # handle 5
#   }
# }

# 2. 删除指定规则
nft delete rule inet filter input handle 5
```

### 方法2：按规则内容删除

```bash
# 删除完全匹配的规则
nft delete rule inet filter input udp dport 53 drop
```

### 删除整条链

```bash
# 删除链（必须先清空规则）
nft flush chain inet filter input
nft delete chain inet filter input
```

### 删除整个表

```bash
# 删除表（会删除表内所有链和规则）
nft delete table ip nat
```

### 清空所有规则

```bash
# 清空所有规则（危险！会断网）
nft flush ruleset
```

## 编辑规则

### 方法1：删除后重新添加

```bash
# 1. 查看规则句柄
nft -a list chain ip nat prerouting

# 2. 删除旧规则
nft delete rule ip nat prerouting handle 3

# 3. 添加新规则
nft add rule ip nat prerouting tcp dport 8080 dnat to 192.168.1.100:80
```

### 方法2：插入规则到指定位置

```bash
# 在句柄 5 之前插入规则
nft insert rule inet filter input position 5 tcp dport 22 accept

# 在句柄 5 之后添加规则
nft add rule inet filter input position 5 tcp dport 80 accept
```

### 方法3：替换规则

```bash
# 替换句柄 5 的规则
nft replace rule inet filter input handle 5 tcp dport 443 accept
```

## 规则持久化

nftables 的规则默认重启后丢失，需要保存。

### 保存规则

```bash
# 保存到配置文件
nft list ruleset > /etc/nftables.conf
```

### 加载规则

```bash
# 从配置文件加载
nft -f /etc/nftables.conf
```

### 开机自动加载

```bash
# 启用 nftables 服务
systemctl enable nftables
systemctl start nftables

# 服务会自动加载 /etc/nftables.conf
```

## 实用技巧

### 备份当前规则

```bash
# 备份
nft list ruleset > ~/nftables-backup-$(date +%Y%m%d).conf

# 恢复
nft flush ruleset
nft -f ~/nftables-backup-20260324.conf
```

### 测试规则（不持久化）

```bash
# 添加临时规则
nft add rule inet filter input tcp dport 8888 drop

# 测试...

# 重启后自动消失，或手动删除
nft delete rule inet filter input handle X
```

### 查看规则统计

```bash
# 显示每条规则的匹配次数
nft list ruleset -a -s
```

### 清空特定表的规则

```bash
# 只清空 nat 表
nft flush table ip nat
```

## 常见操作速查

| 操作 | 命令 |
|------|------|
| 查看所有规则 | `nft list ruleset` |
| 查看带句柄号 | `nft -a list ruleset` |
| 删除规则 | `nft delete rule <表> <链> handle <号>` |
| 清空所有规则 | `nft flush ruleset` |
| 保存规则 | `nft list ruleset > /etc/nftables.conf` |
| 加载规则 | `nft -f /etc/nftables.conf` |
| 删除表 | `nft delete table <表名>` |

## 安全提示

- 删除规则前先备份
- 测试新规则时保持 SSH 连接，避免锁死自己
- 使用 `nft -f` 加载配置前先检查语法
- 重要服务器建议设置定时任务自动清空规则（防止锁死）

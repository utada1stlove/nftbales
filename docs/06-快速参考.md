# nftables 快速参考

## 常用命令速查

### 查看规则

```bash
nft list ruleset              # 查看所有规则
nft -a list ruleset           # 带句柄号查看
nft list table ip nat         # 查看 nat 表
nft list chain inet filter input  # 查看特定链
```

### 删除规则

```bash
nft -a list ruleset           # 先查看句柄号
nft delete rule inet filter input handle 5  # 删除指定规则
nft flush table ip nat        # 清空 nat 表
nft flush ruleset             # 清空所有规则（危险）
```

### 保存和加载

```bash
nft list ruleset > /etc/nftables.conf  # 保存规则
nft -f /etc/nftables.conf              # 加载规则
systemctl enable nftables              # 开机自动加载
```

## 端口转发速查

### 内网IP转发

```bash
# 开启转发
echo 1 > /proc/sys/net/ipv4/ip_forward

# 创建表和链
nft add table ip nat
nft add chain ip nat prerouting { type nat hook prerouting priority -100 \; }
nft add chain ip nat postrouting { type nat hook postrouting priority 100 \; }

# 转发规则（80 -> 192.168.1.100:8080）
nft add rule ip nat prerouting tcp dport 80 dnat to 192.168.1.100:8080
nft add rule ip nat postrouting ip daddr 192.168.1.100 masquerade
```

### 公网IP转发

```bash
# 与内网转发相同，只是目标IP不同
nft add rule ip nat prerouting tcp dport 80 dnat to 5.6.7.8:8080
nft add rule ip nat postrouting ip daddr 5.6.7.8 masquerade
```

## 禁用UDP流量速查

```bash
# 创建表和链
nft add table inet filter
nft add chain inet filter input { type filter hook input priority 0 \; policy accept \; }

# 禁用单个端口
nft add rule inet filter input udp dport 53 drop

# 禁用多个端口
nft add rule inet filter input udp dport { 53, 123 } drop

# 禁用端口范围
nft add rule inet filter input udp dport 10000-20000 drop
```

## 常用规则模板

### 允许SSH

```bash
nft add rule inet filter input tcp dport 22 accept
```

### 允许HTTP/HTTPS

```bash
nft add rule inet filter input tcp dport { 80, 443 } accept
```

### 允许本地回环

```bash
nft add rule inet filter input iif lo accept
```

### 允许已建立的连接

```bash
nft add rule inet filter input ct state established,related accept
```

### 禁止特定IP

```bash
nft add rule inet filter input ip saddr 1.2.3.4 drop
```

### 限制连接速率

```bash
nft add rule inet filter input tcp dport 22 limit rate 10/minute accept
```

## 表和链类型

### 表类型

| 类型 | 用途 |
|------|------|
| `ip` | IPv4 |
| `ip6` | IPv6 |
| `inet` | IPv4 + IPv6 |
| `arp` | ARP |
| `bridge` | 桥接 |

### 链类型

| 类型 | Hook | 用途 |
|------|------|------|
| `filter` | input/output/forward | 过滤流量 |
| `nat` | prerouting/postrouting | 地址转换 |
| `route` | output | 路由决策 |

## 匹配条件

```bash
tcp dport 80              # TCP 目标端口
udp sport 53              # UDP 源端口
ip saddr 1.2.3.4          # 源IP
ip daddr 5.6.7.8          # 目标IP
iifname "eth0"            # 入口网卡
oifname "eth1"            # 出口网卡
tcp dport 80-443          # 端口范围
tcp dport { 80, 443 }     # 多个端口
ct state established      # 连接状态
```

## 动作

```bash
accept                    # 接受
drop                      # 丢弃（静默）
reject                    # 拒绝（返回错误）
dnat to 1.2.3.4:80       # 目标地址转换
snat to 1.2.3.4          # 源地址转换
masquerade               # 自动源地址转换
```

## 故障排查

### 检查IP转发

```bash
cat /proc/sys/net/ipv4/ip_forward  # 应该是 1
```

### 检查规则是否生效

```bash
nft -a list ruleset       # 查看规则
nft list ruleset -s       # 查看匹配统计
```

### 测试端口

```bash
nc -zv IP 端口            # TCP 测试
nc -zvu IP 端口           # UDP 测试
telnet IP 端口            # TCP 测试
```

### 查看连接跟踪

```bash
conntrack -L              # 查看所有连接
conntrack -L -p tcp       # 只看 TCP
```

## 完整示例：基础防火墙

```bash
# 创建表和链
nft add table inet filter
nft add chain inet filter input { type filter hook input priority 0 \; policy drop \; }
nft add chain inet filter forward { type filter hook forward priority 0 \; policy drop \; }
nft add chain inet filter output { type filter hook output priority 0 \; policy accept \; }

# 允许本地回环
nft add rule inet filter input iif lo accept

# 允许已建立的连接
nft add rule inet filter input ct state established,related accept

# 允许 SSH
nft add rule inet filter input tcp dport 22 accept

# 允许 HTTP/HTTPS
nft add rule inet filter input tcp dport { 80, 443 } accept

# 保存
nft list ruleset > /etc/nftables.conf
```

## 紧急恢复

```bash
# 如果锁死了，从控制台执行
nft flush ruleset         # 清空所有规则
systemctl stop nftables   # 停止服务
```
